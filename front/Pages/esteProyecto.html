En este proyecto hemos hecho uso de un monton de herramientas nuevas, lo cual es algo bastante extenso y complicado, pero con practica (como lo es la Homework) se empeiza a tornat mas facil y entendible. A continuacion todas las tecnologias, frameworks o librerias que hemos aprendido y aplicado en el proyecto entero.

HTML,CSS,JAVASCRIPT - FRONT

Obviamente no podia faltar el trio fantastico para el Frpnt-End, En este proyecto profundizamos aun mas con estas tecnologias. Como por ejemplo lo que fue mas eventos con Javascript combinar CSS y HTML aun mas.

Bootstrap - FRONT

El primer Framework con el que trabajamos (creo jksj) el cual ofrece hermosos estilos ya predefinidos, con muchas funcionalidades que para nosotros due un poco complicado, pero nada que la practica facilite, en lo persoal me gusto pero da conflicto al principio.

Express - BACK

Al igual que bootstrap, express fue algo nuevo para todos nosotros, con la diferencia que no es tan complicado de entender y aprender. Ademas de ser la primera vez que interactuamos bastante mas de lleno con el Back-End.

Nodemon - BACK

Probablemente una de las mas importante por no decir que lo es, este levanta el servidor, actualiza el servidor automaticamente cuando hacemos un cambio, y es muy facil de usar.

Middlewares - BACK

Bueno ante todo la seguridad de tener todos nuestros datos validados, bien guardados etc. Para esto no podian faltar los middlewares como lo son Morgan, Cors, tambien los que vienen incorporados con express.

MongoDB - BACK

Finalmente llegamos a la base de datos, en este caso no relacional, como un muy buen comienzo MongoDB es bastante intuitivo, con una sintaxis clara al igual que su interfaz. Muy facil de configurar ya sea con Atlas o Compass.


Colorin colorado otra trasnochada he superado

Ya para terminar(PORFINN). Este proyecto el cual fue muchisimo mas robusto que el anterior sin duda, aprendi bastantes tecnologias, experiencias debugeando 4 horas seguidas, darme a la curiosdida de investigar mas de lo que nos mandaban, me deja con un buen sabor de boca el darme cuenta que aprendi mucho haciendo esto, evidentemente esta son unas pocas tecnologias, frameworks, librerias, que utilizamos ya que todas son mas que necesarias y utiles.
